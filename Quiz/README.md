# RAG理解度テスト問題集

## 概要

このディレクトリには、RAG構築業務に必要な知識を確認するための理解度テスト問題が保存されています。

## 問題の構成

各問題は個別のファイルとして保存され、以下の情報を含みます：
- 問題番号
- カテゴリ
- 難易度（初級/中級/上級）
- 問題文
- 回答欄
- 評価欄

## ファイル命名規則

`YYYYMMDD_HHMMSS_問題番号_カテゴリ.md`

例: `20260126_130358_001_基本概念.md`

## 使用方法

1. 各問題ファイルを開き、問題文を読む
2. 回答欄に回答を記入する
3. 必要に応じて、参考情報に記載されている実装ファイルを確認する
4. 評価欄に理解度評価とコメントを記入する（自己評価または他者評価）

## 理解度レベル

- **レベル1（基礎）**: 用語を説明できる、基本的な流れを理解している
- **レベル2（応用）**: 技術の仕組みを説明できる、手法の違いを理解している
- **レベル3（実践）**: 実装の詳細を説明できる、精度向上の方法を提案できる

## 時系列追跡

同じカテゴリの問題を複数回実施し、理解度の変化を追跡できます。新しいテストを実施する際は、新しい日時でファイルを作成してください。
